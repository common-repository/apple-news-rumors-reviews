=== Apple - News, rumors & reviews ===
Contributors: runejensen
Tags: rss, feed, apple, news, rumors, reviews
Requires at least: 2.7
Tested up to: 2.8.4

Enables a highly configurable widget which shows a specified number of recent Apple news, rumors and/or reviews.

== Description ==

The widget is meant for anyone who wants to show the latest Apple news, rumors and/or reviews without actually having to write the articles. The widget can be configured to show only news & rumors, only reviews or both news & rumors and reviews. You will also be able to specify how many items you want to show, what the title should be and have full control of the list's formatting.

                   
== Installation ==

1. Upload the folder `applelunch-rss` to the `/wp-content/plugins/` directory or search for `Apple news` in `Plugins` > `Add new` and click `Install` to the right of `Apple - News, rumors & reviews`
2. Activate the plugin through the `Plugins` menu in WordPress
